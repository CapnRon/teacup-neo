# Teacup-T41

T41 (QFN96) adaptation of the [Teacup Rev-C](https://github.com/CapnRon/Teacup) T31 development board.

Targets T41NQ (128MB) / T41XQ (256MB) as pin-identical stuffing options; T41LQ (64MB)
possible via 1.8V DDR rail stuffing option.

- `hw/` - KiCad project (based on TeaCup(C)3.3)
- `docs/` - pin maps, adaptation notes
- `reference/` - upstream 3.3 artifacts (netlist, BOM)

Status: WIP - schematic adaptation not started.
